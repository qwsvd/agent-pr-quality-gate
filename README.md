# agent-pr-quality-gate

AI Coding Agent PR Preflight Review and Failure Analysis.

This project studies how AI coding agents handle real GitHub Issues before submitting pull requests.

## Project Goal

The goal is to build a small, auditable, evidence-based workflow for evaluating AI-generated code changes before real open-source contribution.

## Workflow

1. Collect real GitHub Issues
2. Classify whether they are suitable for AI coding agents
3. Use Codex to generate plans or patches
4. Review the diff manually
5. Check tests or manual validation steps
6. Record whether a PR or Issue comment was submitted
7. Analyze failure reasons

## Core Outputs

1. 30+ curated GitHub Issues
2. 5 deep case studies
3. 3 patch files
4. 1 real PR or Issue interaction attempt
5. 1 failure analysis report
6. 1 quality gate evaluation report

## Project Positioning

This project focuses on AI coding agent workflow, human review, diff risk analysis, validation, and open-source contribution preflight checks.
